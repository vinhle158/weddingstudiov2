# Tài liệu STUDIO V2

## Tài liệu hiện hành

Các tài liệu cần đọc khi tiếp quản production nằm tại:

```text
docs/handoffs/production/
```

Thứ tự đọc khuyến nghị:

1. `BEAUTIFUL_STUDIO_V2_FULL_HANDOFF_2026-07-14.md` — bản trình bày dễ đọc, phù hợp để nắm nhanh toàn cảnh.
2. `00_READ_FIRST_STUDIO_V2_FULL_HANDOFF_2026-07-14.md` — bản kỹ thuật chi tiết để agent tiếp quản và đối chiếu.
3. `HANDOFF_CHAT_NATIVE_LOCAL_2026-07-13.md`
4. `PLAN_PRODUCTION_CHAT_CUTOVER_2026-07-14.md`

## Tài liệu lưu trữ

`docs/archive/` chứa báo cáo, worklog, đặc tả và mockup của các giai đoạn cũ. Nội dung trong archive chỉ dùng để tra cứu lịch sử, không được xem là cấu hình production hiện hành.

Khi thông tin giữa archive và handoff production khác nhau, ưu tiên handoff production rồi xác minh lại bằng source, Git và server ở chế độ chỉ đọc.
